A Directory of Clojure Libraries

This is a categorized and annotated directory of available Clojure libraries and tools. This directory is not comprehensive and is necessarily highly opinionated.

This directory is manually curated by the Clojure community. Please endeavor to keep it up-to-date, consisting of high quality libraries with adequate documentation. There are many more libraries in the Clojure ecosystem, but some lack documentation and/or are useful primarily to experienced developers and such projects are not included in this document.

For more comprehensive overview of the Clojure library ecosystem, please see ClojureSphere.

Legend:

  • If a library is listed here with a description and also on clojure-toolbox, I've left it in place here but marked it with three stars (in case Toolbox wants to add summary lines).
  • If a library is listed here and is not on clojure-toolbox but seems useful and well-maintained, I've marked it with three pluses (and will propose to Toolbox shortly).

Note: If a library was listed here without a description and also on clojure-toolbox, I've removed it from here. If a library was listed here but seems abandoned, I've removed it from here.

Applications & Environment

  • *** tools.cli: a command line argument parser for Clojure

  • *** environ: Manage environment settings from a number of different sources

Testing

  • +++ cloverage: a test-coverage tool

  • *** expectations: a minimalist's testing framework (diff URL)

  • +++ test.generative: generative testing, a la QuickCheck

Serialization

JSON

  • *** cheshire: very efficient Clojure JSON and SMILE (binary JSON) encoding/decoding.

  • *** data.json: JSON parser/generator to/from Clojure data structures.

Clojure Reader

  • *** Nippy: a more efficient implementation of the Clojure reader

XML

  • *** data.xml: a library for reading and writing XML

Binary Formats

  • *** gloss: turns complicated byte formats into Clojure data structures -- moved to clj-commons

File formats

  • *** clj-pdf: a library for easily generating PDFs from Clojure

  • *** Pantomime: a tiny Clojure library that deals with Internet media types (MIME types) and content type detection

  • *** data.csv: a CSV parser

Templating

  • *** Stencil: Mustache for Clojure (logic-less templates). Fast.

  • *** Clostache: another nice Mustache implementation

HTTP

Client/Server

  • *** HTTP Kit: High-performance event-driven HTTP client/server for Clojure.

Client

  • *** clj-http: An idiomatic Clojure http client wrapping the apache client.

Logging

  • *** Timbre: Simple, flexible, all-Clojure logging. No XML!

  • *** tools.logging: standard general-purpose logging.

Web Development

Web Services

  • *** Ring: foundational web application library

  • *** Compojure: concise routing library for Ring

  • *** Pedestal: an open source tool set for building web applications in Clojure

  • *** Luminus: lein template for creating batteries-included web applications using Ring, Compojure, lib-noir, and other libraries.

  • *** Liberator: a Clojure library for building RESTful applications

  • *** friend: Authentication and authorization library for Web apps

HTML Generation

  • *** hiccup: Generates HTML from Clojure data structures.

  • *** Stencil: Implements the Mustache templating language.

  • *** markdown-clj: Clojure based Markdown parsers for both Clojure and ClojureScript.

HTML Parsers

  • *** Crouton: A Clojure wrapper for the JSoup HTML and XML parser that handles real world inputs

Data Validation

  • *** Validateur: functional validations library inspired by Ruby's ActiveModel

  • *** Metis: another validations library inspired by Ruby's ActiveModel

URIs, URLs

  • *** route-one: a tiny Clojure library that generates HTTP resource routes (as in Ruby on Rails, Jersey, and so on)

Internationalization (i18n), Localization (l10n)

  • +++ Tower: a simple, idiomatic internationalization and localization story for Clojure

RSS

  • *** clj-rss: RSS feed generation library

Data Stores

Relational Databases, JDBC

  • *** next.jdbc: Modern wrapper for JDBC. Works with all JDBC databases (MySQL, PostgreSQL, Oracle, SQL Server, etc).

CouchDB

  • *** Clutch: Apache CouchDB client.

MongoDB

  • *** Monger: Monger is an idiomatic Clojure MongoDB driver for a more civilized age with solid documentation

  • *** congomongo: Basic wrapper for the MongoDB Java driver

Redis

  • *** Carmine: a great Clojure client for Redis

Graph Databases (Neo4J, Titan, etc)

  • *** Neocons: Neocons is a feature rich idiomatic Clojure client for the Neo4J REST API with solid documentation eprints

ElasticSearch

  • *** Elastisch: Elastisch is a minimalistic Clojure client for ElasticSearch with solid documentation.

Memcached, Couchbase, Kestrel

  • *** Spyglass: Spyglass is a very fast Clojure client for Memcached and Couchbase with solid documentation

Apache Cassandra

  • *** Cassaforte: A young Clojure client for Apache Cassandra

  • *** Alia: Cassandra CQL3 client for Clojure, datastax/java-driver wrapper

Networking

  • ***Lamina: event-driven workflows in Clojure

  • ***Aleph: asynchronous communication in Clojure

Application Servers

  • ***Immutant: a feature rich and integrated application platform for Clojure from Red Hat

Messaging

RabbitMQ

  • ***Langohr: a feature complete RabbitMQ client that embraces AMQP 0.9.1 model and learns from others

Beanstalk

  • ***beanstalk: a Beanstalkd client

Amazon SQS

  • ***Bandalore: a Clojure client library for Amazon's Simple Queue Service

Data Processing, Computation

  • ***Cascalog: data processing on Hadoop without the hassle

Automation, Provisioning, DevOps Tools

  • *** Amazonica: comprehensive Clojure client for the entire AWS API

  • +++ clj-ssh: an SSH client

Monitoring, metrics

  • +++ metrics-clojure: Clojure library on top of Yammer's Metrics

  • +++ clj-statsd: simple client library to interface with statsd

  • +++ riemann: A network event stream processing system, in Clojure.

I/O

Files, NIO, NIO2

File I/O is covered by the JDK and commonly used via clojure.java.io functions.

  • *** fs: utilities for working with the file system

Standard Streams, Subprocesses

  • *** conch: for shelling out to external programs. An alternative to clojure.java.shell.

REPL and Terminal

  • +++ REPLy: a Swiss army knife of interactive editing, and better REPL for Clojure

Mathematics

  • *** math.numeric-tower: various utility math functions

  • +++ math.combinatorics: common combinatorial functions

  • *** core.matrix: matrix operations

Email

  • *** Postal: generate and send email with Clojure

  • *** Mailer: generate and send email using Postal and Moustache templates

Data Structures and Algorithms

Caching

  • *** core.cache: the Clojure API for various cache implementations

Monads

  • *** algo.monads: macros for defining monads, and definition of the most common monads

  • *** protocol-monads: A protocol based monad implementation for clojure

Scheduling

  • *** Quartzite: a powerful scheduling library

Graphics and GUI

  • *** Quil: For making drawings, animations, and artwork (some examples). Wraps the "Processing" graphics environment.

  • *** seesaw: A Swing wrapper/DSL.

  • *** clisk: Clisk is a DSL-based library for procedural image generation that can be used from Clojure and Java.

Security and Sandboxing

  • *** Clojail: a [code execution] sandboxing library

Documentation

Literate Programming

  • *** Marginalia: literate programming implementation for Clojure. See the Marginalia site for an example.

Generating API Reference

  • *** Codox: from the author of Compojure. See compojure api docs for an example.

  • *** Autodoc: used to generate the official Clojure API reference.

Tooling

  • *** Leiningen: the Clojure build tool

  • *** nREPL: nREPL interface

  • +++ java.jmx: nice JMX interface

  • *** tools.trace: a tracing library

  • *** criterium: a benchmarking library that tries to address common benchmarking pitfalls
